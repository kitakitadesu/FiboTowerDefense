# Guide: Presentation

**Core Idea**: Structure and timing for the final project presentation.

**Key Points**:
- Total Time: 6 mins presentation + 10 mins Q&A
- Overview: Quick intro (what the game does)
- Requirements: Show functional/non-functional requirements
- Architecture: Show UML/System overview
- UI: Explain design decisions
