## การนำ OOP ไปใช้

| หลักการ | การนำไปใช้ | ตำแหน่ง |
|---------|-------------|----------|
| **คลาส** | 13+ คลาส: Game, Level, GameBoard, PlacementGrid, HighScoreTable, Tower, Turret, Enemy, Projectile, WaveManager, SolarCell, Scoreboard, Sprite | `include/*.hpp` |
| **สืบทอด** | PlacementGrid : Board, HighScoreTable : Board (2 จุดจาก Board ฐาน) | `include/game_grid.hpp`, `include/high_score_table.hpp` |
| **ห่อหุ้ม** | ฟิลด์ทั้งหมด private พร้อม getters สาธารณะ — ไม่มี data members ที่เปิดเผยในทุกคลาส | ทั่วทั้งโค้ด |
| **ปรับสถานะขณะรันไทม์** | เพิ่ม/ลบศัตรู ป้อม โซลาร์เซลล์จาก vector ขณะรันไทม์ | `Level::update()`, `Level::addTurret()`, dialog ขาย |
